# koboutil

Utilities for Kobo eReader maintenance

- `kobocoverclean`: Given a mounted Kobo path, checks if the image cache is inconsistent and asks if to clean it.
- `kobosearch`: Given a mounted Kobo path and a regular expression, searches for content in the SQLite database.
- `kobocachepurge`: Given a mounted Kobo path and a regular expression, searches for image covers to remove from the device.
- `kobotool`: Given a path, can do all other three functions in one utility.
